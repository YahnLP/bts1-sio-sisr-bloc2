## Objectifs Pédagogiques

**À l'issue de cette séance, l'apprenant sera capable de :**

1. **Identifier** les classes A, B, C d'adresses IPv4 et les plages privées/publiques
2. **Utiliser** la notation CIDR (`/24`, `/16`...) et calculer les plages d'hôtes correspondantes
3. **Expliquer** le fonctionnement du protocole ARP (résolution adresse MAC ↔ IP)
4. **Expliquer** le rôle du protocole ICMP (ping, traceroute, messages d'erreur)
5. **Configurer** manuellement une adresse IP sous Windows (interface graphique)
6. **Configurer** manuellement une adresse IP sous Linux (commandes `ip addr`, `ip route`)
7. **Installer** Debian en machine virtuelle et naviguer dans la ligne de commande Linux
8. **Interpréter** la sortie d'un `ping` et d'un `tracert`/`traceroute`

### Prérequis

- Notion d'adresse IPv4 et masque de sous-réseau (S4)
- Commandes `ping`, `ipconfig` sous Windows (S4)
- Installation de VM (S2 - Windows en VM)
- Conversions binaire/décimal (Maths S1)
